# Wheres The Money?
A program that uses functions to create a financial breakdown of the users monthly salary based on user input. 

## Installation

Simply download vs code and then install the .py file to run the code. 

## Usage

For all questions the function asks, the user should respond accordingly. for example, an accurate response to 
```python
#What is your annual salary? Please input only numbers without commas.
```
would be your annual salary without any extra characters, such as '25000'.

Below is an example of a chart generated for an annual salary of $35,000, with a monthly rent of $2,200, monthly bills that amount to around $500, monthly food expenses that amount to $550, and monthly traveling expenses that are about $300. 

```python
---------------------------------------------------------------------------------------------------------------------------
See the financial breakdown below, based on a salary of $35000
---------------------------------------------------------------------------------------------------------------------------
| mortgage/rent | $  26,400.00 |  75.4% | ###########################################################################
|         bills | $   6,000.00 |  17.1% | #################
|          food | $  28,600.00 |  81.7% | #################################################################################
|        travel | $     300.00 |   0.9% | 
|           tax | $   7,000.00 |  20.0% | ####################
|         extra | $ -33,300.00 | -95.1% | 
---------------------------------------------------------------------------------------------------------------------------
>>> WARNING: DEFICIT <<<
```
